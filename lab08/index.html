I will be explaining 5 different form elements

A form is an HTML element used to accept information from website visitors

Form Elements

  • Scrolling Text Box Form

    A scrolling text box form accepts free-form comments or questions. This element configures a scrolling text box.

  • Select List Form

    The select list form, also known as a dropdown list, is used to configure one select element and multiple option elements.

  • Radio Button Form

    The radio button form control allows the user to select exactly one choice from the provided options.

    Option 2
  • Check Box

    The checkbox form is used to configure checkboxes, allowing website visitors to select one or more options.



  • Password Box Form

    The password box form control is used to keep information hidden. It configures a password that users have to enter.
