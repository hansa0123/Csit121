What is XML? How is XML used? Examples of XML
XML, or Extensible Markup Language, is a markup language designed for storing and transporting data. It is a simple and flexible text-based format that allows users to define their own customized tags for describing data. JavaScript (JS)

What is JavaScript? How is JavaScript related to HTML and CSS? Common JavaScript elements, properties, methods, and functions
JavaScript is a scripting language that enables interactive web pages. It adds behavior and interactivity to web pages. JavaScript is often used in conjunction with HTML and CSS. HTML provides the structure, CSS handles the presentation, and JavaScript adds behavior.
  • Elements: Variables, Data Types, Arrays, Objects
  • Properties: Length, innerHTML
  • Methods: String Methods, Array Methods, Object Methods
  • Functions: Declaration, Arrow Functions, Callback Functions, Anonymous Functions
